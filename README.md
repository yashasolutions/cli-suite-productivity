- bash 
- gdrive (https://github.com/prasmussen/gdrive)

## commands

- `lf` : list last n files 

### Google Drive 
- `glink` 	: turn ID in into gdrive link 
- `glist` 	: list files from gdrive 
- `gshare` 	: share file to gdrive and share link + copy to clipboard
